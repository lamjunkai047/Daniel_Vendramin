Description:
Procuramos um freelancer experiente para desenvolver uma solução de previsão de vendas utilizando o modelo Prophet da Meta. O projeto envolve a criação de um código em Python ou uma aplicação que possa ser executada mensalmente para gerar novas previsões de vendas. A base de dados disponível contém informações históricas de vendas de diversos produtos de um fabricante, detalhadas mês a mês, ao longo de vários anos.

Os principais objetivos do projeto incluem:
- Implementar o modelo Prophet para prever vendas futuras com base nos dados históricos fornecidos.
- Desenvolver um script Python robusto e fácil de usar, ou uma aplicação, que permita a atualização mensal das previsões.
- Incluir uma funcionalidade para comparar o desempenho das previsões geradas pelo modelo Prophet com as previsões manuais dos vendedores (input manual). Esta comparação é crucial para avaliar a acurácia de ambos os métodos e identificar qual deles oferece maior precisão.

Esperamos uma solução que seja eficiente, escalável e que forneça insights claros sobre o desempenho das vendas futuras e a eficácia dos diferentes métodos de previsão. O freelancer deve ter forte conhecimento em Python, modelagem de dados e experiência com bibliotecas de machine learning, especialmente Prophet.
